## 🌟 About the Project

This repository contains the source code for my **personal developer portfolio website**. Built using modern web technologies (**React.js**, **Vite**, and **CSS3**), it delivers a fast, responsive, and aesthetically pleasing user experience to showcase my technical projects, certifications, and experience.

---

## ✨ Features

- 🎨 **Modern & Responsive UI**: Glassmorphic elements, vibrant themes, and smooth micro-animations.
- ⚡ **Lightning Fast Performance**: Powered by Vite and optimized asset loading.
- 📱 **Cross-Device Compatibility**: Seamless experience across mobile, tablet, laptop, and desktop displays.
- 💻 **Interactive Project Cards**: Direct links to live deployments and GitHub repositories.
- 📜 **Certificate Viewer**: High-resolution interactive certificate previews.
- 📂 **Clean Codebase**: Modular React components following best practices.

## 🚀 Local Development Setup

Follow these steps to run the portfolio locally on your machine:

1. **Clone the repository**
   ```bash
   git clone https://github.com/aniruddhadhar7/Portfolio.git
   ```

2. **Navigate into the project directory**
   ```bash
   cd Portfolio
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the local development server**
   ```bash
   npm run dev
   ```

5. **Open in browser**  
   Navigate to `http://localhost:5173` to view the running app.
