# 🌟 About the Project

This repository contains the source code for my **personal portfolio website**, developed to showcase my technical skills, projects, certifications, achievements, and experience.

The portfolio is designed with a clean, responsive, and modern interface to provide visitors with a seamless browsing experience across all devices.

# 📁 Folder Description

## 📂 assets

Contains all static resources such as images, icons, logos, and other media files used throughout the application.

---

## 📂 components

Contains reusable React components that build different sections of the portfolio, making the project modular and maintainable.

---

## 📄 App.jsx

The main component responsible for rendering the overall application layout.

---

## 📄 App.css

Contains styling specific to the main application layout.

---

## 📄 main.jsx

Entry point of the React application where the root component is rendered.

---

## 📄 index.css

Contains global styles applied throughout the project.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/aniruddhadhar7/Portfolio.git
```

Navigate into the project

```bash
cd Portfolio
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```
